# Biblioteca-Inteligente-Agente-Reactivo
Biblioteca Inteligente - Agente Reactivo Simple
Descripción

Este proyecto implementa un agente reactivo simple para la gestión de una biblioteca inteligente utilizando Java Swing. El sistema permite registrar usuarios, administrar libros y gestionar préstamos mediante reglas de percepción–acción.

Funcionalidades
Registro de usuarios.
Registro de libros.
Gestión de préstamos.
Reserva automática de libros sin disponibilidad.
Historial de decisiones del agente.
Estadísticas de uso.
Arquitectura del Agente
Percepciones
usuario_nuevo
libro_registrado
libro_disponible
libro_no_disponible
libro_vencido
Acciones
Registrar usuario
Agregar libro
Prestar libro
Reservar libro
Generar multa
Tecnologías utilizadas
Java
Swing
JTable
JTabbedPane
Ejecución
javac BibliotecaInteligentePro.java
java BibliotecaInteligentePro
